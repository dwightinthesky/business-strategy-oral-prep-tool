# Linear Strategy Research Deck

A Linear-inspired, dark-theme interactive research site for **企業組織與策略導論**.

## What is inside

- 10 chapter research map based on the full report
- Deep-dive summary, framework cards, and case analysis per chapter
- Oral exam drill mode with random questions and answer scaffolds
- Built-in keyword search for models, cases, and concepts

## Tech

- Single-page static site
- Vue 3 (CDN)
- Custom CSS (Linear style visual language)

## Local preview

Open `index.html` in a browser.

## Deploy

Designed for Cloudflare Pages static deployment.
